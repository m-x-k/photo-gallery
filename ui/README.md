UI photo gallery
================

* Frontend-UI: React, React Photo Gallery, npm, webpack

Requires:
---------

* npm (e.g. brew install npm)
* webpack (e.g. npm install webpack -g)

Setup:
------

```npm install```

Run:
----

```npm start```

http://localhost:8080/

Package:
--------

Run the command ```webpack``` in order to produce 'index.js'
